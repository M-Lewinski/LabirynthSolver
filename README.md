# LabirynthSolver
Labirynth solver written in OCaml.

## Plik wejściowy

Plik wejściowy powinien przedstawiać schemat labiryntu o następujących oznaczeniach:
- '-': ściana
- 'S': start
- 'E': koniec
- '0': ścieżka

W pierwszej lini musi znajdować się szerokość labiryntu a w kolejnej jego wysokość.
Przykład:
6
6
```
S00-
--0-
000-
--0E
```
